# Dark Night Theme

Custom dark theme for VS Code in the Neon style.

## Features

- Neon colors: pink, blue, purple
- Optimized for Next.js, TypeScript, Tailwind CSS
- Pleasant contrast for long-term development

## Installation

1. Open VS Code
2. Press `Ctrl+K Ctrl+T`
3. Select `Dark Night`
